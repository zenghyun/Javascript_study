## Different Kinds of Tests

- Unit Tests
   - Fully Isolated ( e.g. testing one function )

<br>

- Integration Tests
  - With Dependencies ( e.g. testing a function that calls a function )
   
<br>


- End-to-End (E2E) Tests 
  - Full Flow ( e.g. validating a the DOM after a click)
